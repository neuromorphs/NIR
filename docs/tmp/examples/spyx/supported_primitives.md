### Supported Primitives in Spyx

This library supports conversion of the following nodes to NIR:
- Conv2d
- Affine
- Linear
- CubaLIF
- IF
- LI
- LIF

This library supports conversion of the following nodes from NIR:
- Conv1d
- Conv2d
- Delay
- Flatten
- Affine
- Linear
- Scale
- CubaLIF
- I
- IF
- LIF
- SumPool2d
- Threshold
